💳 Thiết Lập Thanh Toán — SePay VietQR

Mục tiêu: Phát hiện thanh toán tự động và biết khách mua gói nào
1
Đăng ký SePay & lấy API
  • Vào sepay.vn → Đăng ký tài khoản → Xác minh CMND
  • Dashboard → API → Copy API Token
  • Thêm tài khoản ngân hàng → Bật "Nhận webhook thanh toán"
# Cấu hình webhook URL (điền sau khi setup Make.com) Webhook URL: https://hook.make.com/YOUR_HOOK_ID Sự kiện: Nhận tiền thành công
2
Quy ước nội dung chuyển khoản
Khách phải ghi đúng nội dung CK — đây là cách hệ thống nhận biết gói nào:
BRONZE [SĐT khách] → VD: BRONZE 0901234567 SILVER [SĐT khách] → VD: SILVER 0901234567 GOLD [SĐT khách] → VD: GOLD 0901234567
💡 Hướng dẫn rõ trên sales page — "Ghi đúng nội dung để nhận kit tự động"
3
Chuẩn bị 3 link Google Drive ZIP
  • Nén folder goi-bronze/ → BDS-Kit-Bronze.zip → Upload Drive → Lấy link "Anyone with link"
  • Nén folder goi-silver/ → BDS-Kit-Silver.zip → Upload Drive → Lấy link
  • Nén folder goi-gold/ (template, chưa cá nhân hóa) → Upload Drive → Dùng để tham khảo
# Lưu 3 link này để điền vào Make.com BRONZE_LINK = https://drive.google.com/file/d/XXX SILVER_LINK = https://drive.google.com/file/d/YYY
🛠️ Công cụ dùng: ✓ SePay — miễn phí ✓ Google Drive — miễn phí 15GB
2

🔗 Kết Nối Tự Động — Make.com (Miễn Phí)

Make.com nhận webhook từ SePay → tự động gửi email + Telegram cho anh/chị
1
Tạo tài khoản Make.com
  • Vào make.com → Sign up miễn phí
  • Free tier: 1.000 operations/tháng — đủ cho ~200 đơn/tháng
  • Tạo Scenario mới → chọn trigger: Webhooks → Custom Webhook
  • Copy URL webhook → Dán vào SePay dashboard
2
Cấu hình Scenario — Logic phân gói
Luồng Make.com (kéo thả, không cần code):
── TRIGGER ────────────────────────────── Webhook nhận từ SePay └─ content: "BRONZE 0901234567" └─ amount: 1990000 └─ bankAccountName: "NGUYEN VAN A" ── ROUTER (phân loại) ─────────────────── If content CHỨA "BRONZE" → Route 1 (Bronze) If content CHỨA "SILVER" → Route 2 (Silver) If content CHỨA "GOLD" → Route 3 (Gold) ── ROUTE 1: BRONZE ────────────────────── 1. Gmail → Gửi email (template Bronze) 2. Telegram → Thông báo anh/chị ── ROUTE 2: SILVER ────────────────────── 1. Gmail → Gửi email (template Silver + Calendly) 2. Telegram → Thông báo anh/chị ── ROUTE 3: GOLD ──────────────────────── 1. Gmail → Gửi email yêu cầu thông tin team 2. Telegram → Thông báo GOLD ưu tiên
3
Kết nối Gmail trong Make.com
  • Trong Make.com → Add module: Gmail → Send an Email
  • Connect Gmail account → Cho phép quyền truy cập
  • Để gửi từ tên cửa hàng: Dùng Brevo (cũ là Sendinblue) — miễn phí 300 email/ngày, gửi từ tên miền riêng
4
Kết nối Telegram Bot
  • Trong Make.com → Add module: Telegram Bot → Send a Message
  • Nhập Token Bot (đã tạo theo goi-gold/telegram-bot-tu-dong.html)
  • Chat ID: ID Telegram cá nhân của anh/chị
# Template thông báo Gold (ưu tiên) 🏆 ĐƠN GOLD — XỬ LÝ NGAY 👤 Khách: {{bankAccountName}} 📞 SĐT: {{extractedPhone}} 💰 Số tiền: {{amount}}đ 🕐 Lúc: {{transactionDate}} ⚡ Cần nhận thông tin 5 thành viên → Zalo khách ngay
🛠️ Công cụ dùng: ✓ Make.com — miễn phí 1000 ops/tháng ✓ Gmail — miễn phí ✓ Brevo — miễn phí 300 email/ngày ✓ Telegram Bot — miễn phí
