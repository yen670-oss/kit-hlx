Phase 1 — Chuẩn Bị File & Link

Tạo ZIP cho từng gói + link Calendly để nhúng vào email

⏱ ~20 phút
1

Tạo 3 file ZIP — một ZIP cho mỗi gói

Nén riêng từng gói thành file ZIP để gửi qua Google Drive

  1. Mở Windows Explorer → vào thư mục AI Agent Business Kit
  2. Chuột phải thư mục goi-bronze → Send to → Compressed (zipped) folder
  3. Đặt tên: Kit-BDS-HalongXanh-BRONZE.zip
  4. Lặp lại với goi-silver → Kit-BDS-HalongXanh-SILVER.zip
  5. Lặp lại với goi-gold → Kit-BDS-HalongXanh-GOLD.zip
✅
Tên file không dấu, không space — tránh lỗi khi khách tải về trên Mac/Windows
2

Upload 3 ZIP lên Google Drive & lấy link chia sẻ

Mỗi gói cần 1 link public riêng

  1. Vào drive.google.com → Tạo thư mục mới: Kit BDS Delivery
  2. Upload 3 file ZIP vào thư mục này
  3. Chuột phải mỗi file ZIP → Share → Get link → Anyone with the link
  4. Copy từng link, lưu vào notepad:
📝 Notepad — lưu 3 link này
BRONZE_LINK = https://drive.google.com/file/d/XXXX/view?usp=sharing SILVER_LINK = https://drive.google.com/file/d/YYYY/view?usp=sharing GOLD_LINK = https://drive.google.com/file/d/ZZZZ/view?usp=sharing
💡
Đổi link từ /view?usp=sharing sang /uc?export=download để khách nhấn → tải thẳng không qua preview
3

Tạo tài khoản Calendly (cho gói Silver & Gold)

Link đặt lịch tư vấn 60 phút, nhúng vào email giao hàng

  1. Vào calendly.com → Sign up miễn phí bằng Gmail
  2. Tạo Event: New Event Type → One-on-One
  3. Điền thông tin:
FieldĐiền vào
Event nameTư vấn BĐS Hạ Long Xanh — Miễn phí 60 phút
Duration60 minutes
LocationZalo / Google Meet (anh chị chọn)
DescriptionPhân tích ROI cá nhân hóa cho ngân sách của bạn
Available hoursT2-T6: 9h-17h, T7: 9h-12h
  1. Save → Copy link Calendly (dạng calendly.com/ten-cua-ban/tu-van-60p)
  2. Lưu vào notepad: CALENDLY_LINK = ...
✅
Calendly free plan đủ dùng — 1 event type, kết nối được Google Calendar để block lịch tự động

Phase 2 — SePay: Nhận Thanh Toán Tự Động

Kết nối tài khoản ngân hàng + tạo webhook gửi sang Make.com

⏱ ~20 phút
1

Tạo tài khoản SePay

Đăng ký miễn phí tại sepay.vn

  1. Vào sepay.vn → Đăng ký → Xác thực số điện thoại
  2. Vào Tài khoản ngân hàng → Thêm tài khoản
  3. Chọn ngân hàng của anh/chị → Nhập số tài khoản + tên chủ tài khoản
  4. Làm theo hướng dẫn xác thực quyền sở hữu (SePay sẽ chuyển 1.000đ để xác nhận)
⚠️
Chỉ cần 1 tài khoản ngân hàng — SePay đọc lịch sử SMS/Internet Banking để detect giao dịch
2

Tạo QR Code cho 3 gói

Mỗi gói 1 QR riêng — để Make.com phân biệt được BRONZE / SILVER / GOLD

🔑
Quy tắc quan trọng: Nội dung chuyển khoản phải chứa từ khoá BRONZE/SILVER/GOLD. Hướng dẫn khách điền đúng trong nội dung CK.
  1. Vào SePay → Tạo QR VietQR
  2. Tạo QR cho gói BRONZE:
FieldGói BRONZEGói SILVERGói GOLD
Số tiền99000019900002990000
Nội dung CK BRONZE [SĐT] SILVER [SĐT] GOLD [SĐT]
Tên giao dịchKit BronzeKit SilverKit Gold
  1. Download 3 ảnh QR → nhúng vào landing page và email báo giá
💡
Hướng dẫn khách: "Anh/chị điền nội dung CK: BRONZE + số điện thoại anh/chị" → Make.com sẽ tự detect
3

Lấy Webhook URL từ SePay

URL này sẽ điền vào Make.com ở Phase 4

  1. SePay Dashboard → Tích hợp → Webhook
  2. Chọn ngân hàng đã kết nối → Thêm Webhook
  3. Để trống URL — điền sau khi có Make.com URL từ Phase 4
  4. Lưu ý Webhook sẽ gửi JSON dạng:
JSON payload từ SePay
{ "id": 1234, "gateway": "VCB", "transactionDate": "2025-05-15 14:30:00", "accountNumber": "1234567890", "code": null, "content": "BRONZE 0901234567", "transferType": "in", "transferAmount": 990000, "accumulated": 990000, "referenceCode": "REF001", "description": "...", "subAccount": null, "bankAccount": "..." }
📌
Field quan trọng nhất: "content" — chứa nội dung chuyển khoản. Make.com sẽ đọc field này để route.

Phase 3 — Telegram Bot: Thông Báo Tức Thì

Tạo bot Telegram nhận alert mỗi khi có đơn mới

⏱ ~15 phút
1

Tạo Bot qua BotFather

BotFather là bot chính thức của Telegram để tạo bot

  1. Mở Telegram → tìm kiếm @BotFather → Start
  2. Gõ lệnh: /newbot
  3. Đặt tên hiển thị: Kit BDS Alert
  4. Đặt username (phải kết thúc bằng bot): kitbdshalongbot
  5. BotFather sẽ gửi lại HTTP API Token — dạng:
Bot Token (ví dụ — của anh/chị sẽ khác)
7234156789:AAHbQ8Kl2mNpXvRsYz0tWuJqGdCeFiHoAkLm
  1. Lưu token này vào notepad: BOT_TOKEN = ...
🔒
Token này = mật khẩu bot — không share công khai. Nếu lộ thì vào BotFather → /revoke để tạo token mới.
2

Lấy Chat ID của anh/chị

Make.com cần Chat ID để gửi tin nhắn đến đúng người

  1. Tìm kiếm bot vừa tạo (tên username anh/chị đặt) → Start
  2. Gõ bất kỳ tin nhắn gì: hello
  3. Mở trình duyệt, vào URL sau (thay YOUR_TOKEN):
https://api.telegram.org/botYOUR_TOKEN/getUpdates
  1. Trang web hiện ra JSON → tìm field "id" trong phần "chat":
Tìm số này trong JSON
"chat": { "id": 123456789, "first_name": "Tên Của Bạn", "type": "private" }
  1. Lưu vào notepad: CHAT_ID = 123456789
💡
Nếu muốn alert vào group Telegram (cả team nhận): tạo group → add bot vào → lấy Chat ID của group (thường số âm, ví dụ: -1001234567890)
3

Test gửi tin nhắn thử

Xác nhận Bot Token + Chat ID hoạt động trước khi dùng

  1. Mở URL sau trên trình duyệt (thay token và chat_id):
https://api.telegram.org/botYOUR_TOKEN/sendMessage?chat_id=YOUR_CHAT_ID&text=Bot+hoat+dong+tot!
  1. Nếu Telegram nhận được tin "Bot hoat dong tot!" → ✅ Phase 3 hoàn thành
  2. Lưu cả 2 giá trị: BOT_TOKEN và CHAT_ID
📱
Xem thêm hướng dẫn chi tiết Telegram Bot tại: telegram-bot-tu-dong.html (trong goi-gold)

Phase 4 — Make.com: Trung Tâm Điều Phối

Xây Scenario: Webhook → Router → Email + Telegram (phần phức tạp nhất)

⏱ ~60 phút
🗺️
Luồng logic tổng thể:
TRIGGER: SePay Webhook POST
├─ Đọc field "content" trong JSON
│
ROUTER:
IF content CONTAINS "BRONZE" → Route 1
IF content CONTAINS "SILVER" → Route 2
IF content CONTAINS "GOLD"    → Route 3
│
Route 1 (BRONZE): Gmail giao hàng + Telegram alert
Route 2 (SILVER): Gmail giao hàng + Calendly + Telegram alert
Route 3 (GOLD):   Gmail hỏi thông tin team + Telegram GOLD alert
1

Tạo tài khoản Make.com & Scenario mới

Free plan: 1,000 operations/tháng (~200 đơn hàng)

  1. Vào make.com → Sign up miễn phí bằng Gmail
  2. Dashboard → Create a new scenario
  3. Đặt tên: Kit BDS — Auto Delivery
  4. Click + để thêm module đầu tiên
2

Module 1: Webhooks — Custom Webhook (TRIGGER)

Nhận dữ liệu từ SePay khi có giao dịch mới

  1. Tìm kiếm: Webhooks → chọn Custom webhook
  2. Click Add → Đặt tên: SePay Payment Trigger
  3. Make.com tạo ra URL webhook — dạng:
Webhook URL (ví dụ)
https://hook.make.com/abc123xyz456def789
  1. Copy URL này → Quay lại SePay → Paste vào field Webhook URL → Save
  2. Quay lại Make.com → Click Run once (để Make.com chờ nhận data)
  3. Thực hiện 1 chuyển khoản test qua SePay (ít nhất 1,000đ) với nội dung TEST 0000
  4. Make.com nhận data → hiện Data structure detected → Click Save
✅
Sau bước này Make.com đã biết cấu trúc JSON từ SePay. Anh/chị sẽ thấy các field như content, transferAmount, transactionDate...
3
