⚡ Cách nhanh hơn: Git Command Line
Upload tất cả 1 lệnh — kể cả folder txt · Cần cài git trước

Nếu đã cài git, đây là cách upload tất cả file (kể cả txt) chỉ bằng 4 lệnh:

✅
Kiểm tra git: Mở CMD → gõ git --version → nếu thấy số version là OK. Chưa có → tải tại git-scm.com
CMD / Terminal — Chạy lần lượt 
# Mở CMD, paste từng lệnh

cd "C:\Users\ADMIN\OneDrive\LƯU TRỮ KÊNH\AI Agent Business Kit\kit-ban-kit"

git init

git remote add origin https://github.com/yen670-oss/kit-hlx.git

git add *.html 02-CONTENT-BAN-KIT/ 04-XU-LY-TINH-HUONG/

git commit -m "upload full kit"

git branch -M main

git push -u origin main --force
🔑
Git sẽ hỏi username (yen670-oss) và password → dùng Personal Access Token thay mật khẩu. Tạo token: GitHub → Settings → Developer settings → Personal access tokens → Generate new → chọn scope repo → Generate → Copy token.
🔄
Lần sau chỉ cần 3 lệnh khi muốn update:
git add . → git commit -m "update" → git push

Câu Hỏi Thường Gặp
Xử lý nhanh khi gặp lỗi
Bình thường — GitHub tự hỏi có muốn ghi đè không. Click "Commit changes" bình thường. File cũ sẽ bị thay thế bởi file mới.
Cần đợi 2–5 phút sau khi commit. GitHub Pages cần thời gian build. Nếu sau 5 phút vẫn 404 → vào Settings → Pages → kiểm tra branch đang là main và folder là / (root).
File index.html là file mặc định cho đường dẫn gốc. Kiểm tra repo có file index.html ở root (không nằm trong subfolder). Sales page hiện tại đang đúng vị trí.
Chạy lệnh này thay thế: git remote set-url origin https://github.com/yen670-oss/kit-hlx.git rồi tiếp tục với git push -u origin main --force.
GitHub đã tắt xác thực bằng mật khẩu thường. Cần dùng Personal Access Token. Tạo tại: github.com → Settings → Developer settings → Personal access tokens → Generate new token → scope: repo → Generate → Copy token. Khi git hỏi password → paste token này (không thấy ký tự nhưng đã nhập).
Cách web UI: GitHub → repo → click file cần update → Edit (bút chì) → paste nội dung mới → Commit.

Cách git: Sửa file → git add ten-file.html → git commit -m "update" → git push. Link live tự cập nhật sau ~2 phút.
