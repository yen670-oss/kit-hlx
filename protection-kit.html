🛡️ PROTECTION KIT · MODULE TÁI SỬ DỤNG
Bộ Bảo Vệ Tài Sản Số
5 lớp bảo vệ · Copy-paste vào bất kỳ dự án HTML nào · Không phụ thuộc framework
Cách sử dụng module này: Chọn từng snippet bên trái → Copy → Paste vào dự án mới. Mỗi snippet độc lập, không phụ thuộc nhau. Có thể dùng tất cả hoặc từng phần tuỳ nhu cầu.
1
🏷️ Copyright Meta — Paste vào <head>
Khai báo tác giả và bản quyền cho tất cả trang. Google và các bot sẽ nhận diện quyền sở hữu. Phát sinh hiệu lực pháp lý ngay từ khi deploy.
2
📄 Footer Copyright — Paste trước </body>
Hiển thị thông báo bản quyền cho người dùng. Kèm link Terms of Use. Deterrent tâm lý — người thấy thường do dự trước khi copy.
3
🚫 Anti-Copy JS — Dành cho Sales/Pitch page
Disable chuột phải, Ctrl+U/S/A, F12, phát hiện DevTools. Dùng cho trang sales, không dùng cho hub admin (sẽ block cả anh dùng).
4
🔐 Token Gate — Dành cho Delivery/Gated page
Chặn truy cập trang giao kit nếu không có token hợp lệ. Mỗi khách nhận 1 token riêng qua Token Manager. Redirect về sales page nếu không có token.
5
⚡ Batch Script — Áp dụng hàng loạt
PowerShell script xử lý toàn bộ folder HTML cùng lúc. Copy script → chạy 1 lần → tất cả file được bảo vệ. Dùng khi có project mới với nhiều file.
Dùng nhanh: Dự án mới → Chạy Batch Script (layer 1+2) → Thêm Anti-Copy vào sales page (layer 3) → Thêm Token Gate vào delivery page (layer 4). Xong. Mỗi layer mất <5 phút.

🚫 Anti-Copy Full Protection JS
Vô hiệu hóa: chuột phải · Ctrl+U/S/A · F12 · DevTools. Khi phát hiện DevTools → xóa trắng nội dung trang và hiện thông báo bảo vệ.
<!-- ===== ANTI-COPY PROTECTION ===== -->
<script>
(function(){
  // 1. Disable right-click
  document.addEventListener('contextmenu', function(e){
    e.preventDefault();
  });

  // 2. Disable keyboard shortcuts
  document.addEventListener('keydown', function(e){
    // Block: Ctrl+U (source), Ctrl+S (save), Ctrl+A (select all)
    if(e.ctrlKey && ['u','s','a'].includes(e.key.toLowerCase())){
      e.preventDefault(); return false;
    }
    // Block: F12, Ctrl+Shift+I/J/C (DevTools)
    if(e.key === 'F12'){ e.preventDefault(); return false; }
    if(e.ctrlKey && e.shiftKey && ['i','j','c'].includes(e.key.toLowerCase())){
      e.preventDefault(); return false;
    }
  });

  // 3. Detect DevTools open → clear page
  var devtools = { open: false };
  setInterval(function(){
    var t = new Date();
    debugger; // Pauses if DevTools open
    if(new Date() - t > 100 && !devtools.open){
      devtools.open = true;
      document.body.innerHTML =
        '<div style="display:flex;align-items:center;justify-content:center;' +
        'height:100vh;background:#0a1628;color:#E2C06A;' +
        'font-family:sans-serif;text-align:center;padding:40px">' +
        '<div><div style="font-size:48px;margin-bottom:20px">🛡️</div>' +
        '<div style="font-size:22px;font-weight:bold;margin-bottom:12px">' +
        'Nội dung được bảo vệ</div>' +
        '<div style="font-size:13px;opacity:.6">' +
        '© 2026 [THƯƠNG HIỆU] · [TÊN TÁC GIẢ]<br>' +
        'Nghiêm cấm sao chép dưới mọi hình thức</div>' +
        '</div></div>';
    }
  }, 1000);
})();
</script>

🔐 SNIPPET 4 · JAVASCRIPT
Token Gate — Bảo Vệ Trang Giao Kit
Paste vào <head> sau charset. Người không có token hợp lệ sẽ bị redirect về sales page. Mỗi khách nhận 1 link riêng kèm token.
Cách dùng: Định nghĩa danh sách VALID_TOKENS → deploy → Khi giao kit, gửi khách link dạng page.html?t=TOKEN123
🔐 Token Gate — Template Tái Sử Dụng JS
Thay SALES_PAGE_URL và VALID_TOKENS theo dự án. Mỗi token là 1 chuỗi unique — có thể dùng tên khách, mã đơn hàng, hoặc code random.
<!-- ===== TOKEN GATE ===== -->
<script>
(function(){
  // ─── CONFIG: Thay đổi theo dự án ─────────────────────
  var SALES_PAGE = 'ten-sales-page.html'; // Redirect nếu không có token
  var VALID = [
    'TOKEN001', 'TOKEN002', 'TOKEN003',
    // Thêm token tại đây...
  ];
  // ─────────────────────────────────────────────────────

  var params = new URLSearchParams(window.location.search);
  var token  = (params.get('t') || params.get('token') || '').toUpperCase();
  var session = sessionStorage.getItem('kit_access');

  // Skip gate for localhost/file (admin preview)
  var host = window.location.hostname;
  var isLocal = (host === 'localhost' || host === '' || host === '127.0.0.1');

  if(!isLocal && !session && !VALID.includes(token)){
    window.location.href = SALES_PAGE;
    return;
  }
  if(token) sessionStorage.setItem('kit_access', '1');
})();
</script>

⚡ CÔNG CỤ · POWERSHELL
Batch Script — Áp Dụng Hàng Loạt
Copy script → Mở PowerShell → Paste → Enter. Tự động thêm copyright meta + footer vào tất cả file HTML trong folder chỉ định.
Cách chạy: Nhấn Win + X → Windows PowerShell → Paste script → Thay đường dẫn → Enter
⚡ Batch Copyright Script — PowerShell PowerShell
Xử lý tất cả .html trong folder. Tự động skip file đã có copyright. Thay $dir, $author, $year, $brand theo dự án mới.
# ─── CONFIG ───────────────────────────────────────
$dir    = "C:\Users\ADMIN\Desktop\TEN-DU-AN"  # Thay đường dẫn
$author = "[TÊN TÁC GIẢ]"
$brand  = "[THƯƠNG HIỆU]"
$year   = "2026"
# ──────────────────────────────────────────────────

$metaBlock = @"

  <meta name="author" content="$brand · $author">
  <meta name="copyright" content="© $year $brand · $author. Bản quyền được bảo hộ.">
"@

$footerHtml = @"

<div style="background:rgba(0,0,0,.5);border-top:1px solid rgba(255,255,255,.06);
  padding:14px 20px;text-align:center;font-size:11px;
  font-family:sans-serif;color:rgba(255,255,255,.25)">
  © $year $brand · $author · Nghiêm cấm sao chép dưới mọi hình thức.<br>
  <a href="terms.html" style="color:rgba(201,168,76,.4);text-decoration:none">
    Điều Khoản Sử Dụng</a>
</div>
"@

$files = Get-ChildItem -Path $dir -Filter "*.html"
$done = 0

foreach ($file in $files) {
    $content = [System.IO.File]::ReadAllText($file.FullName, [System.Text.Encoding]::UTF8)
    $changed = $false

    if ($content -notmatch 'name="copyright"') {
        $content = $content -replace '(<meta charset=[''"]UTF-8[''"]>)', "`$1$metaBlock"
        $changed = $true
    }
    if ($content -notmatch 'Nghiêm cấm sao chép') {
        $content = $content -replace '</body>', "$footerHtml`n</body>"
        $changed = $true
    }
    if ($changed) {
        [System.IO.File]::WriteAllText($file.FullName, $content, [System.Text.Encoding]::UTF8)
        $done++
    }
}
Write-Host "✅ Hoàn tất: $done files đã được bảo vệ"
Chạy: PowerShell (Win+X) · Thay $dir trước khi chạy
