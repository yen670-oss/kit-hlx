Quy trình giao link kit cho khách sau thanh toán:
Bước 1: Chọn token chưa dùng (status: FREE) đúng gói khách mua
Bước 2: Click 📋 Copy Link → tự động tạo URL có token
Bước 3: Nhắn link qua Zalo cho khách, ghi chú tên khách vào ô Note
Bước 4: Click ✓ Đánh dấu đã dùng để theo dõi
💡 Lưu ý: Mỗi token chỉ dùng cho 1 khách. Token bị chia sẻ → vô hiệu hóa không hoàn lại.
Tất cả thay đổi được lưu vào localStorage của trình duyệt này.
