Link Live Trên Feed
FB & Zalo — Từng Bước Chi Tiết

Tạo thumbnail 1200×630, cài OG tags, cách đăng bài để link hiện preview card clickable ngay trên bảng feed — không cần chạy quảng cáo.

🖼 Thumbnail tải về ngay
📋 Copy-paste ready

🖼 Bước 1 — Tạo Thumbnail OG Image (1200×630px)

✏️ Tùy chỉnh thumbnail

⚠️
Sau khi tải thumbnail về

Đổi tên file thành og-image.png → bỏ vào thư mục 01-SALES-PAGE/ → deploy lại Netlify.

🏷 Bước 2 — Cập Nhật OG Tags Trong Sales Page

1

Mở file 01-SALES-PAGE/index.html bằng Notepad / VS Code

Nhấn Ctrl+H (Find & Replace) → tìm đoạn THAY-LINK-NETLIFY-CUA-ANH-CHI

2

Thay bằng link Netlify thật của anh/chị

Ví dụ link sau khi deploy là https://kit-hlx.netlify.app thì thay tất cả chỗ đó.

Sau khi thay, 3 dòng OG quan trọng sẽ trông như này:

<!-- Thay TOÀN BỘ "THAY-LINK..." bằng link Netlify thật -->

<meta property="og:url"
      content="https://kit-hlx.netlify.app">

<meta property="og:image"
      content="https://kit-hlx.netlify.app/og-image.png">

<meta name="twitter:image"
      content="https://kit-hlx.netlify.app/og-image.png">
3

Bỏ file og-image.png vào thư mục 01-SALES-PAGE/

Thumbnail vừa tải về ở Bước 1 → đổi tên thành og-image.png → copy vào cùng thư mục với index.html.

Cấu trúc thư mục sau khi xong:

01-SALES-PAGE/
├── index.html         ← Sales page chính
└── og-image.png       ← Thumbnail vừa tạo ✅
4

Re-deploy lên Netlify Bắt buộc

Vào app.netlify.com → chọn site → tab Deploys → kéo thả lại thư mục 01-SALES-PAGE/ → chờ 30 giây.

URL không thay đổi — Netlify tự cập nhật nội dung mới.

🔄 Bước 3 — Reset Cache Facebook (Không Bỏ Qua)

⚠️
Vì sao phải reset cache?

Facebook lưu cache thông tin link trong nhiều ngày. Nếu bỏ qua bước này, khi đăng bài link vẫn hiện ảnh cũ / không có ảnh dù anh/chị đã cập nhật og:image.

1

Vào Facebook Sharing Debugger

Truy cập: developers.facebook.com/tools/debug/

Đăng nhập bằng tài khoản Facebook cá nhân (không cần tài khoản developer).

2

Dán link Netlify → Bấm "Debug"

Dán đường link đầy đủ của sales page vào ô trắng → bấm nút Debug.

FB sẽ đọc OG tags của trang và hiển thị preview.

3

Bấm "Scrape Again" 1–2 lần

Sau khi kết quả hiện ra → bấm nút "Scrape Again" (nút màu xanh lá ở cuối trang).

Làm 1–2 lần cho chắc. FB sẽ cập nhật ảnh thumbnail mới nhất.

✅
Dấu hiệu thành công

Ở phần "Link Preview" cuối trang thấy ảnh og-image.png hiện đúng + tiêu đề + mô tả → done!

📢 Bước 4 — Kỹ Thuật Đăng Bài Để Link Hiện Live

📘
Facebook
Cá nhân · Fanpage · Group
  • 1Tạo bài mới → click vào ô "Bạn đang nghĩ gì?"
  • 2Dán link Netlify vào ô soạn bài
  • 3Chờ 3–5 giây → preview card tự hiện (ảnh + tiêu đề)
  • 4Xóa URL khỏi phần text — card vẫn giữ nguyên
  • 5Gõ nội dung bài viết bên trên card
  • 6Đăng bình thường
Xóa URL text → feed trông chuyên nghiệp hơn. Khách thấy card ảnh đẹp → bấm thẳng vào trang.
💬
Zalo
Cá nhân · OA · Group
  • 1Zalo cá nhân: Dán link vào nhật ký / tin nhắn → Zalo tự show preview → gửi bình thường
  • 2Zalo OA: Vào trang OA → Tạo "Bài viết" → Paste link → chọn dạng Article → thêm nút CTA "Đăng ký ngay"
  • 3Group Zalo: Dán link vào ô chat nhóm → Zalo preview tự hiện → gửi
Zalo OA có nút CTA tích hợp (Đăng ký · Mua ngay · Tìm hiểu thêm) — conversion rate cao hơn 40% so với link thuần.
Cách đăng Feed hiện gì Click-through Nên dùng
❌ Chỉ dán URL text Link màu xanh thuần Thấp — 1–2% Không
⚠️ Paste link + không xóa text Card + URL text lộ dài Trung bình — 3–5% Được, không tối ưu
✅ Paste link → xóa text → đăng Card đẹp: ảnh + tiêu đề + nút Cao — 8–15% ✅ Khuyến nghị
🚀 FB Ads với link preview Card + sponsored badge Rất cao — 15–25% Khi có ngân sách

📋 Bước 5 — OG Tags Đầy Đủ Để Copy Vào <head>

ℹ️
Khi nào cần dùng đoạn này?

Nếu anh/chị có landing page khác (không phải file kit cung cấp), copy đoạn dưới đây vào bên trong thẻ <head>...</head> của trang đó.

<!-- ═══ OG TAGS — dán vào trong <head> ═══ -->

<!-- Cơ bản -->
<meta property="og:type"       content="website">
<meta property="og:locale"     content="vi_VN">

<!-- Thay 3 chỗ dưới bằng thông tin thật -->
<meta property="og:url"        content="https://zingy-fox-d95ab1.netlify.app/">
<meta property="og:title"      content="BDS Mastery Kit — Bộ Vũ Khí Bán Hàng Vinhomes HLX">
<meta property="og:description" content="Landing page live 3 phút · 30 bài viết sẵn đăng · Lead tự động 24/7.">

<!-- Ảnh thumbnail (1200×630px) -->
<meta property="og:image"       content="https://zingy-fox-d95ab1.netlify.app/og-image.png">
<meta property="og:image:width"  content="1200">
<meta property="og:image:height" content="630">
<meta property="og:image:alt"    content="BDS Mastery Kit — Kit bán hàng HLX">

<!-- Twitter/X card -->
<meta name="twitter:card"        value="summary_large_image">
<meta name="twitter:title"       content="BDS Mastery Kit — Bộ Vũ Khí Bán Hàng HLX">
<meta name="twitter:description"  content="Landing page · 30 bài viết · Lead tự động 24/7">
<meta name="twitter:image"       content="https://[LINK-NETLIFY-CUA-ANH].netlify.app/og-image.png">
